# Purely Works Brand Assets

This folder contains all brand assets, guidelines, and templates for creating on-brand materials.

## Folder Structure

```
brand/
├── colors/               # Color palette and usage guidelines
│   └── palette.md       # Complete color specifications
├── logos/               # Logo files (SVG, PNG, variations)
├── fonts/               # Brand typography files
├── templates/           # Branded templates
│   ├── email/          # Email signatures and templates
│   ├── proposal/       # Proposal document templates
│   ├── presentation/   # Slide deck templates
│   └── document/       # Word doc and report templates
├── assets/             # Supporting brand assets (icons, patterns, imagery)
└── guidelines/         # Brand usage guidelines and standards
```

## What's Needed

To complete this brand folder, please provide:

### Logos (Critical)
- [ ] Primary logo (full color, SVG or high-res PNG)
- [ ] Logo variations (white version, icon only, horizontal/stacked)
- [ ] Favicon or icon mark

### Typography (Important)
- [ ] Primary font family name (and files if custom)
- [ ] Heading font (if different from body)
- [ ] Fallback web-safe fonts

### Templates (Nice to Have)
- [ ] Existing proposal template (if any)
- [ ] Slide deck template (if any)
- [ ] Email signature HTML (if standardized)

### Additional Assets (Optional)
- [ ] Photography style guide or sample images
- [ ] Icon set (if using custom icons)
- [ ] Patterns or textures used in designs
- [ ] Partner/client logos for case studies
