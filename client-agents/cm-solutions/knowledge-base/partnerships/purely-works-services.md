## TECHNICAL ARCHITECTURE SUMMARY

### Data Flow Diagram (Conceptual)

```
Indeed/LinkedIn → Make.com Automation
                     ↓
                 Softr Platform
                 (Resume Upload)
                     ↓
            AI Scoring Engine
                     ↓
          Salesforce CRM (Primary)
                 ↓ & ↑
            Airtable Database
          (Talent Pool Search)
                     ↓
              Notifications
            (Email alerts)
```

### System Integration Points

1. **External Job Boards → Make.com:** Pulls job postings and applications
2. **Make.com → Softr:** Sends candidate data to web platform
3. **Softr → Salesforce:** Auto-creates records, AI scoring applied
4. **Salesforce → Airtable:** Two-way sync for talent pool management
5. **Airtable → Email System:** Sends notifications and communications
6. **Make.com → LinkedIn/Indeed:** Bidirectional data flow
