# VSCE Brand Assets & Guidelines

## Overview

This folder contains VSCE's brand assets, design guidelines, and templates for maintaining consistent professional presentation across all client-facing materials, proposals, and communications.

---

## Brand Asset Inventory

### Logos (`/brand/logos/`)
- **VSCE Logo (Primary)** — Horizontal lockup, full color
- **VSCE Logo (Secondary)** — Vertical stacked version
- **VSCE Monochrome** — Black and white versions (for restricted color environments)
- **Icon/Mark** — Simplified icon for small applications
- **Certification Badges** — DBE, SBE, MBE, SLEB, LBE logos
- **Partner Logos** — Purely Works and other strategic partners

**Logo Usage Guidelines:**
- Minimum size: 1" at 300 DPI
- Clear space: 1/4 of logo height on all sides
- Never modify, distort, or alter the logo
- Use primary color logo on light backgrounds
- Use white logo on dark backgrounds
- Use black/gray logo on colored backgrounds

### Color Palette (`/brand/colors/`)
- **Primary Colors**
  - Corporate Blue: #003A70 (professional, authoritative)
  - Accent Teal: #008B8B (energy, innovation)
  - Dark Gray: #333333 (text, depth)

- **Secondary Colors**
  - Light Gray: #F5F5F5 (backgrounds, spacers)
  - White: #FFFFFF (primary background)
  - Accent Orange: #FF8C00 (highlights, CTAs) — use sparingly

- **Usage Guidelines:**
  - Corporate Blue for primary elements, headers, backgrounds
  - Teal accent for highlights and calls-to-action
  - Dark gray for body text and details
  - Light gray for section backgrounds and dividers
  - White for primary backgrounds and contrast

### Typography (`/brand/fonts/`)
- **Primary Font (Headlines):** Montserrat or similar modern sans-serif
  - Bold (700) for main headlines
  - SemiBold (600) for subheadings
  - Usage: H1, H2, section headers

- **Secondary Font (Body):** Open Sans or Roboto
  - Regular (400) for body text
  - SemiBold (600) for emphasis within text
  - Usage: Body copy, descriptions, details

- **Monospace (Code/Technical):** Courier New or Monaco (if technical content needed)

**Font Size Guidelines:**
- H1 Headers: 32-40pt (headlines, major titles)
- H2 Subheaders: 24-28pt (section headers)
- H3 Sub-subheaders: 18-20pt (subsection titles)
- Body Text: 10-12pt (proposals, documents)
- Captions: 8-9pt (figure captions, footnotes)

## Brand Usage Standards

### General Principles
1. **Consistency** — Apply brand elements consistently across all materials
2. **Professionalism** — Maintain professional government consulting aesthetic
3. **Simplicity** — Use clean design without excessive ornamentation
4. **Alignment** — Align with transportation/government sector standards
5. **Flexibility** — Adapt templates as needed while maintaining core identity

### Government Sector Aesthetic
- Professional, formal appearance
- Minimal use of trendy design elements
- Evidence-based, data-focused visual approach
- Accessibility and readability focus
- Conservative color palette
- Clear hierarchical information structure

### What Reflects VSCE Brand Well
- Professional government consulting expertise
- Specialized transportation focus
- Small, responsive, trusted advisory firm
- Boutique quality with scale efficiency
- Innovation in process and service delivery
- Diverse business enterprise commitment

### What Doesn't Fit VSCE Brand
- Trendy or youthful design (not our market)
- Flashy or excessive colors/graphics
- Corporate/bureaucratic coldness
- Commoditized generic consulting
- Overly complex or hard-to-understand visuals
- Lack of strategic positioning

## Using the Templates

### InDesign Proposal Template (Umair Khalid)

**To Use:**
1. Open master InDesign template file
2. Save as new document with project name
3. Customize cover with project title and client
4. Import proposal content into body pages
5. Adjust graphics, charts, and visuals
6. Apply consistent styling throughout
7. Export to PDF for distribution

**Maintaining:**
- Umair Khalid updates as needed
- Keep master template clean and unchanged
- Save working files as new versions
- Archive final proposals in project folder

### PowerPoint Presentation Template

**To Use:**
1. Open template file
2. Save with presentation title
3. Add content using pre-designed slide layouts
4. Maintain consistent color and typography
5. Use graphics and charts library
6. Apply VSCE logo on title and closing slides

## Compliance & Certification Logos

### Proper Display of Certifications

**DBE (Caltrans)**
- Use official Caltrans DBE logo
- Size: Minimum 1" x 1"
- Display on proposals and marketing (when pursuing Caltrans work)
- Include in qualifications section

**SBE (CA DGS)**
- Use official state SBE logo
- Include on proposals for state projects

**MBE (CA PUC)**
- Use official MBE logo
- Display when pursuing utility/PUC projects

**SLEB (Alameda County)**
- Use official county SLEB logo
- Display on Bay Area proposals

**LBE (Port of Oakland & Others)**
- Use official Port/agency LBE logo
- Display on relevant project proposals

**LA Metro Certifications (DBE/SB/LBE)**
- Use LA Metro official logos
- Include on all LA Metro proposals

**Best Practice:**
- Include certification logos on cover page and/or qualifications section
- Keep certifications current and valid
- Verify certification status before each proposal
- Display with brief statement about what certification means
